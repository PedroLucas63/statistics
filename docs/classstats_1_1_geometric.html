Descrição detalhada

Uma classe para representar uma distribuição geométrica.

A distribuição geométrica é uma distribuição de probabilidade discreta que representa a probabilidade de realizar N tentativas antes de obter sucesso.

◆ getProbability()

double stats::Geometric::getProbability ( int replicasMade) const
inlinevirtual

Calcula a probabilidade de um dado número de tentativas antes de obter o primeiro sucesso.

A probabilidade de um dado número de tentativas antes de obter o primeiro sucesso é dada pela probabilidade de erro (1 - probabilidade de sucesso) elevado ao número de tentativas menos 1, vezes a probabilidade de sucesso.

Parâmetros
replicasMadeO número de tentativas feitas.
Retorna
A probabilidade de um dado número de tentativas antes de obter o primeiro sucesso.
