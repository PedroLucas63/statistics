Descrição detalhada

Uma classe para representar uma distribuição binomial.

A distribuição binomial é uma distribuição de probabilidade discreta que representa o número de sucessos em um certo número de tentativas.

Veja também
https://en.wikipedia.org/wiki/Binomial_distribution
